### Exercise 1: Variable Declaration and Initialization

- Create initial character variables

  1. `+` operator

  3. `String.Concat()` method

- Use `$""` syntax for string interpolation
- Format numbers with specific decimal places
- Include conditional expressions in interpolations
- Create multi-line verbatim strings with `@""`

### Exercise 5: String Methods

- Use `Trim()` to remove whitespace
- Use `Replace()` for text substitution
- Use `Contains()`, `IndexOf()`, `StartsWith()`, `EndsWith()` for searching
- Process messy data into clean, formatted output
